This blog

This blog is built with Jekyll, uses a slighty modified version of the Hyde theme and is hosted by Github Pages. All the content and code is open source.

Want to take a look, or suggest changes? Check out or fork the Github project.

Syntax highlighting is done with Pygments. The stylesheet for syntax highlighting uses the vim desert theme. The css version is generated by uraimo.
