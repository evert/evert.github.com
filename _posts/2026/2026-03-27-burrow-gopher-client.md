After building a [gopher server and website][1], I couldn't quite get enough
of the [Gopher][2] bug, and decided to build a Gopher client in the browser.

To be honest, this is actually more like a HTTP proxy for Gopher, but I might
at some point turn this into a standalone client or even port it to some
native platforms.

Anyway, here's a screenshot:

<figure>
    <img src="/assets/posts/gopher/burrow.png" alt="Screenshot of Burrow, a Gopher client in the browser">
    <figcaption>Burrow screenshot</figcaption>
</figure>

And if you want to try it out, go to [https://burrow.din.gy/](https://burrow.din.gy/),
and browse Gopherspace!

It's GPL licensed, source code is available on [GitHub][3].

Visit my website, at `gopher://hole.din.gy/` !
